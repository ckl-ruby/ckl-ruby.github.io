顺序表

定义一个data数组来存储线性表的所有元素，再定义一个整型变量length来存储线性表的实际长度，用结构体SqList表示如下：

typedef struct
{ ElemType data[MaxSize];
int length;
} SqList; //顺序表类型

(注:ElemType是自定义类型语句，例如定义为int类型可用 ==typedef int ElemType==)

    单链表

单链表中的结点类型LinkNode定义如下：

typedef struct LNode //定义单链表结点类型
{ ElemType data;
struct LNode *next; //指向后继结点
} LinkNode;

插入和删除结点是单链表中最常见的的两种操作：
插入：1.s->next = p->next;

      2. p->next = s;

删除：1.q=p->next;

  1. p->next=q->next;
  2. free(q);

    双链表

双链表中的结点类型DLinkNode定义如下：

typedef struct DNode
{
ElemType data;
struct DNode prior;
struct DNode next;
} DLinkNode;

在双链表p 结点后插入 s的操作如下：
s->next=p->next;
p->next->prior=s;
s->prior=p;
p->next=s;

在p结点后删除 s的操作如下：
p->next->next->prior = p;
p->next = p->next->next;

    循环链表
